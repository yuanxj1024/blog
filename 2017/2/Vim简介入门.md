# Vim简介入门

###基础

* 模式
 * 编辑模式， 俗称vim模式
 * 插入模式
 * 命令模式

 
* **切换模式** 
 * 进入vim模式：默认键是`ESC`
 * 进入插入模式（有如下几个按键）：
     * `i` - 当前位置插入
     * `s`
     * `a`
     * `x`
 * 进入命令模式：使用`:` 冒号

* 基本移动
 * H - 向**前**一个字符移动
 * J - 向**下**一行移动
 * K - 向**上**一行移动
 * L - 向**后**一个字符移动
