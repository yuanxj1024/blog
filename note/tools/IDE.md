# IDE 快捷键

# VS Code 
| 按键 | 说明 |
| --- | ---|
cmd + alt + k  |  添加书签 |
cmd + k + 0    | 代码全部折叠 |
Ctrl + -  |  上次访问的代码位置 |
Ctrl  + \] |   跳转到定义 |
